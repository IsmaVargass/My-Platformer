# Mi juego 2D Plataforma

Proyecto de juego de plataformas 2D creado por Ismael Vargas. 

## Características
- Movimiento fluido del jugador.
- Sistema de Doble Salto (limitado a 2).
- Múltiples niveles.
- Fondos de bosque y nuevos personajes.

## Controles
- **A / D** o **Flechas**: Moverse.
- **Espacio**: Saltar (máximo 2 veces).
- **Móvil**: Joystick y botón de salto.
